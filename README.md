Crucible Creation Grouping
Clean ancestry & background grouping for Crucible
Crucible Creation Grouping adds collapsible, full‑width headers to the Ancestry and Background selection menus in Crucible’s Hero Creation screen.
This makes large compendiums easier to navigate and organizes ancestries/backgrounds by their source packs.

✨ Features
Automatic grouping by compendium

Clean Crucible‑style headers

Collapsible sections

Chevron indicators (▶ / ▼)

No layout distortion — uses Crucible’s native grid

Fully compatible with all Crucible ancestry/background packs

Translation‑ready (lang/en.json included)
